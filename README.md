# 🌾 Crop Recommendation System

## 📌 Project Overview

The Crop Recommendation System is a Machine Learning-based web application that recommends the most suitable crop based on soil nutrients and environmental conditions. It helps farmers and agricultural professionals make informed decisions to improve crop yield and optimize resource utilization.

The application is built using Python, Scikit-learn, and Streamlit, providing a simple and interactive interface for users.

---

## 🚀 Features

* 🌱 Predicts the most suitable crop
* 🤖 Machine Learning-based prediction
* 📊 Interactive Streamlit web interface
* ⚡ Fast and accurate recommendations
* 🌍 User-friendly design

---

## 🧠 Machine Learning Model

* **Programming Language:** Python

## 📊 Input Features

The model predicts the best crop using the following features:

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature (°C)
* Humidity (%)
* Soil pH
* Rainfall (mm)

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Crop-Recommendation.git
```

Navigate to the project directory:

```bash
cd Crop-Recommendation
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
